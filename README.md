# Weather App

A weather application built with React and API integration.

## Demo
[Live Demo](https://weather-app-react-2026.netlify.app/)

## Features
- Search weather by city
- Get weather using current location (Geolocation API)
- Dynamic background based on weather conditions
- Loading state and error handling
- Displays temperature, feels like, humidity, wind, and weather icon
- Clean UI

## Technical Notes
- Built with React (useState, useEffect)
- API requests handled using Axios
- Environment variables used for API key
- Conditional rendering for loading and error states
- Weather-based UI styling using dynamic classNames
- Geolocation support with latitude/longitude handling
